Shell Company Simulator

The Shell Company Simulator is an educational platform that models how legitimate companies and shell entities operate within financial systems. It generates realistic, interconnected data to help visualize ownership structures, transaction behavior, and risk indicators.

What the Generator Does

The simulator programmatically creates:

  • Companies across multiple countries and industries
  • Shell-like entities with offshore locations, minimal staff, and abnormal financial patterns
  • Linked directors and addresses to form ownership networks
  • Financial transactions that mimic real-world money flows
  • A risk score based on structural and behavioral red flags

All data is fully synthetic and used for learning and analytical exploration.

Why This Project Exists

This project was built to make complex concepts like shell companies, AML risk, and corporate opacity easier to understand through simulation. Instead of relying only on real-world leaks or opaque datasets, it provides a controlled environment to observe, test, and explain how suspicious corporate behavior can emerge.
